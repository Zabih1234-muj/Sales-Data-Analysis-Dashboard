# 📊 Sales Data Analysis Dashboard

Interactive dashboard to explore and analyze sales data using **Python**, **Streamlit**, **Pandas**, and **Plotly**.

## Features

- Filter sales by **Region** and **Category**.
- Key metrics: **Total Sales**, **Total Orders**, and **Profit**.
- Interactive charts: **Bar chart**, **Pie chart**, **Line chart (sales trend)**.
- Preview raw dataset.
- Download filtered dataset as CSV.
- Friendly, human-designed interface.
